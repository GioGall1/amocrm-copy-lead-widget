# amoCRM Copy Lead Widget

Виджет для amoCRM, который позволяет копировать сделку в другую воронку и на другой этап одним кликом.

### Основные возможности
- Получение данных исходной сделки через `/api/v4/leads/{id}`
- Создание копии сделки в выбранной воронке и статусе
- Копирование тегов и кастомных полей
- Перенос связанных контактов
- Информирование пользователя о результате (успех / ошибка)

### Используемые технологии
- JavaScript (ES5, совместимый с amoCRM виджетами)
- jQuery (стандарт AMO)
- amoCRM API v4
- Асинхронные цепочки Promises

### Установка
Виджет распространяется в ZIP-архиве и устанавливается через amoMarket → Приватные интеграции.

### Структура
```
widget/
 ├─ script.js
 ├─ manifest.json
 ├─ i18n/
 └─ images/
```
